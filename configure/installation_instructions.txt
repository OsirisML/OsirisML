Manual Installation

Installing nprint on Debian Linux:

Dependencies:

sudo apt-get install libpcap-dev

sudo apt install g++

sudo apt install make

sudo apt update

tar -xvf nprint-1.2.1.tar.gz

cd nprint-1.2.1

./configure && make && sudo make install

See https://github.com/nprint/nprint for more information
